# Pocketeer - Premium Affiliate Showcase

A blazing-fast, scalable Node.js eCommerce showcase designed to maximize affiliate conversions.

## Features
- **Ultra-Fast Performance:** Built with an In-Memory Cache and Hotwire Turbo to deliver instant, SPA-like page transitions with zero database latency.
- **Robust Backend:** Node.js and Express.js architecture.
- **Cloud Database:** Integrated with MongoDB Atlas for persistent storage of products, reviews, blogs, and settings.
- **Media Hosting:** Integrated with Cloudinary for seamless image uploads via the Admin dashboard.
- **Responsive Design:** Beautiful, premium UI with Dark Mode support and smooth CSS animations.
- **Admin Dashboard:** Secure CMS to effortlessly manage products, blogs, and site-wide settings.
- **Conversion Optimized:** Includes a smart randomizer popup to increase buyer urgency.

## Tech Stack
- **Frontend:** Vanilla JS, CSS3, EJS, Hotwire Turbo, InstantPage
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas, Mongoose
- **Cloud:** Render.com, Cloudinary

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file with `MONGO_URI`, `CLOUDINARY_URL`, `ADMIN_USER`, and `ADMIN_PASS`.
4. Run `npm start`.
